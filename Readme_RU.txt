Модуль написан на Windows Media Foundation API, нужен для быстрого извлечения кадров из видео.
Тестировалось на видео h264 .mp4

Доп. возможности:
  Знает фпс видео и его длительность в секундах
  
Если немножко дописать, то можно возвращать кадры прямо в RAM,
  через экземпляры класса Gdiplus::Bitmap
